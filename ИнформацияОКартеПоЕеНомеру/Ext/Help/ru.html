Обработка Информация о карте по её номеру

Обработка предназначена для получения информации о карте (банк, бренд, логотип и другое) с использованием сервиса https://cardinfo.online/

Порядок работы

  1. Регистрируемся на сайте https://cardinfo.online/. Для этого просто надо указать e-mail на который придет API key;
  2. В поле API Key указываем полученный ключ (ключ вида a40a6346264170b9c79126460d1d457b);
  3. В поле Номер карты укажите номер карты - достаточно первых 6 цифр. При выполнении запроса они все равно обрежутся. Допустимо указание номера с пробелами, например: 4276 6900;
  4. Нажмите кнопку Получить. В блоке Информация о карте будет отображена вся доступная информация.

Другие поля являются необязательными.

В таблице полей, если строка содержит URL, то двойной клик откроет ссылку ассоциированным браузером
